# PRD 001 - Bootstrap do Repositório

## Resumo

Restaurar e normalizar a base do projeto para uma entrega pública, executável e testável do desafio DeveloperStore.

## Requisitos

- `REQ-001`: o repositório deve ser público e utilizável na URL oficial do desafio.
- `REQ-002`: a solução deve usar `src/`, `tests/` e `README.md` na raiz.
- `REQ-003`: o backend deve rodar em `.NET 10` e `C# 14`.
- `REQ-004`: o projeto deve conter instruções claras de setup, execução, migrations e testes.
- `REQ-005`: a governança `ai-spec` deve existir no repositório com PRDs rastreáveis.
