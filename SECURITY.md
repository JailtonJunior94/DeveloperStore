# Security Policy

## Supported Versions

Only the latest release of the DeveloperStore Sales API is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of our application seriously. If you discover any security vulnerability, please report it responsibly:

1. **Do not** open a public issue on GitHub.
2. Send an email to the repository owner/maintainer (e.g. at the contact email provided on the GitHub profile).
3. Include detailed steps, payloads, or proof-of-concept scripts to help us reproduce and understand the vulnerability.

We will acknowledge receipt of your report within 48 hours and work with you to analyze and remediate the issue promptly before public disclosure.
